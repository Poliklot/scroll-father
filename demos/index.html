Scroll UI without framework lock-in

Скролл, который не бесит пользователей и не ломает разработчикам вечер.

Scroll Father собирает повседневные scroll-паттерны в маленький TypeScript toolkit: smooth anchors, scrollspy, reveal, progress bar, direction tracking и infinite loading с cleanup-first API.

Установить Смотреть демо
  • 0 runtime dependencies
  • SSR-safe
  • cleanup-first
  • TypeScript types
landing.ts 
import {
	initAnchorScroll,
	initScrollSpy,
	initRevealOnScroll,
	initScrollProgress,
} from 'scroll-father';

const offset = () => header.offsetHeight + 24;

const cleanups = [
	initAnchorScroll({ offset, updateHash: 'replace' }),
	initScrollSpy({ sections: 'section[id]', offset }),
	initRevealOnScroll({ elements: '[data-reveal]' }),
	initScrollProgress({ cssVariable: '--page-progress' }),
];

Зачем это людям

Пять типичных scroll-болей закрываются одним маленьким пакетом.

01

Fixed header съедает якорь

Динамический offset считает реальную высоту шапки перед каждым переходом.

02

Меню не показывает контекст

ScrollSpy подсвечивает активную секцию и ставит `aria-current` для доступности.

03

Анимации расползаются по проекту

Reveal engine только управляет состоянием, а дизайн остается в CSS.

04

Нужен reading progress

Progress пишется в CSS variable или callback без привязки к UI-фреймворку.

05

Infinite scroll должен уметь abort, retry и done

Sentinel loader держит состояния загрузки, отменяет запросы при cleanup и не требует тяжелых зависимостей.

Toolkit

Не компонентная библиотека. А набор точных DOM-примитивов.

Подключается в vanilla, React, Vue, Astro, Webflow-export или любом месте, где есть DOM. Каждый initializer возвращает cleanup-функцию.

Anchor Scroll

Плавные якоря, focus target, hash modes, delegated links.

initAnchorScroll({ offset })

ScrollSpy

Активное меню, `data-active-section`, `aria-current`.

initScrollSpy({ sections })

Reveal

State engine для появления элементов с stagger и reduced motion.

initRevealOnScroll({ once: true })

Progress

Scroll progress в CSS variable, attribute или callback.

initScrollProgress()

Infinite Loader

Sentinel-based загрузка с page, signal, done, error и retry.

initInfiniteLoader({ sentinel })

Legacy Helpers

Debounce, scroll state, direction tracking и совместимость старого API.

smootherAllAnchorLinks()

Recipes

Сценарии, которые чаще всего нужны в реальных интерфейсах.
