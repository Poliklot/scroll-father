Прокрутка без привязки к фреймворку

Скролл, который не бесит пользователей и не ломает разработчикам вечер.

Scroll Father закрывает повседневные задачи прокрутки: плавные якоря, активное меню, появление элементов, прогресс чтения, направление скролла и бесконечную ленту с аккуратной очисткой.

Установить Смотреть демо
  • без зависимостей
  • безопасно для SSR
  • очистка после инициализации
  • типы TypeScript
landing.ts 
import {
	initAnchorScroll,
	initScrollSpy,
	initRevealOnScroll,
	initScrollProgress,
} from 'scroll-father';

const offset = () => header.offsetHeight + 24;

const cleanups = [
	initAnchorScroll({ offset, updateHash: 'replace' }),
	initScrollSpy({ sections: 'section[id]', offset }),
	initRevealOnScroll({ elements: '[data-reveal]' }),
	initScrollProgress({ cssVariable: '--page-progress' }),
];

Зачем это людям

Пять типичных scroll-болей закрываются одним маленьким пакетом.

01

Липкая шапка съедает якорь

Динамический отступ считает реальную высоту шапки перед каждым переходом.

02

Меню не показывает контекст

ScrollSpy подсвечивает активную секцию и ставит `aria-current` для доступности.

03

Анимации расползаются по проекту

Reveal engine только управляет состоянием, а дизайн остается в CSS.

04

Нужен reading progress

Progress пишется в CSS variable или callback без привязки к UI-фреймворку.

05

Бесконечная лента должна уметь отмену, повтор и завершение

Загрузчик по маяку держит состояния, отменяет запросы при очистке и не требует тяжелых зависимостей.

Набор инструментов

Не компонентная библиотека. А набор точных DOM-примитивов.

Подключается в обычный JavaScript, React, Vue, Astro, экспорт из Webflow или любое место, где есть DOM. Каждая функция инициализации возвращает функцию очистки.

Плавные якоря

Отступ для шапки, фокус на целевом блоке, управление адресной строкой и делегирование кликов.

initAnchorScroll({ offset })

Активная навигация

Подсветка пункта меню, `data-active-section` и `aria-current` для доступности.

initScrollSpy({ sections })

Появление элементов

Состояния для CSS-анимаций, быстрый stagger и уважение `prefers-reduced-motion`.

initRevealOnScroll({ once: true })

Прогресс чтения

Значение прогресса в CSS-переменной, атрибуте или обработчике.

initScrollProgress()

Бесконечная лента

Загрузка по маяку с номером страницы, сигналом отмены, завершением и повтором.

initInfiniteLoader({ sentinel })

Совместимость

Отложенные обработчики, состояние скролла, направление движения и старые функции для мягкой миграции.

smootherAllAnchorLinks()

Recipes

Сценарии, которые чаще всего нужны в реальных интерфейсах.

Лендинг

Липкая шапка + плавные якоря

initAnchorScroll({
	offset: () => header.offsetHeight + 24,

Документация

Активная навигация документации

initScrollSpy({
	sections: 'section[id]',
	navLinks: '.site-nav a',
	ariaCurrent: 'location',
});
Каталог

Дозагрузка карточек по маяку

initInfiniteLoader({

	loadMore: ({ page, signal, done }) => fetchPage(page, signal, done),
});
